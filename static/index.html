streamit :)

to upstream a file
curl --upload-file myfile.gz streamit.jit.su/chooseApath
to downstream a file
curl streamit.jit.su/chooseApath > myfile.gz
files are never stored on the server. instead, when you upload a file, it pauses the transfer until someone requests to download it.
once someone connects on the other end, it begins streaming the file.
if you try to download from a path where someone is not uploading, it will wait until someone starts uploading to that specific path.
